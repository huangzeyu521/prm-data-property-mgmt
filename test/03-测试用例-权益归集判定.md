# 测试用例：权益归集判定引擎（分子公司共享网公司）

> 规则依据：《数据权益内部管理汇总表》说明页 5 条判定规则。
> 判定输入：管制属性（管制业务/非管制）、是否涉第三方（来源判定非A 或 G/H/I/J 任一为"是"）、申请方经营权（权属类型含"经营"）。

## 规则矩阵

| 规则 | 条件 | 网公司经营权 | 用例 |
|---|---|---|---|
| 1.1 | 自行生产 + 不涉三方 + 管制 | **有**（调整为有，归集网公司） | TC-03-01 |
| 1.2 | 自行生产 + 不涉三方 + 非管制 | 同步申请方（有→有/无→无） | TC-03-02 |
| 2.1 | 涉三方 + 非管制 + 有经营权 | 依权益转移判定 | TC-03-03 |
| 2.2 | 涉三方 + 非管制 + 无经营权 | **无** | TC-03-04 |
| 3.1 | 涉三方 + 管制 | 恢复经营权后：有 I/J 约束→无；否则依权益判定 | TC-03-05/06 |

## TC-03-01 规则1.1（黔电小智样例）【P0】

- 数据：管制业务；表级清单 1 行（日程会议室表/工作秘密/A 自行生产/G-J 全否）
- 步骤：暂存申请并保存清单 → 查看步骤3"权益归集判定"
- 预期：命中规则 1.1；网公司持有/使用/经营=有/有/**有**；原因含"管制单位经营权调整为有,归集至网公司"
- 自动化：`ConfirmConsolidationTest.rule_1_1_self_produced_regulated_consolidates_operate_right`

## TC-03-02 规则1.2 【P0】

- 数据：非管制；权属类型含经营权；A 自行生产；G-J 全否
- 预期：规则 1.2；经营权=有（相应权利同步网公司）
- 自动化：`rule_1_2_self_produced_unregulated_syncs_rights`

## TC-03-03 规则2.1 【P1】

- 数据：非管制；权属类型含经营权；来源判定 B 公开采集（涉三方）
- 预期：规则 2.1；经营权="依权益转移判定"（黄色标签，需人工权益转移判定）

## TC-03-04 规则2.2 【P0】

- 数据：非管制；权属类型仅持有权；来源判定 B 公开采集
- 预期：规则 2.2；网公司经营权=**无**；involvesThird=true
- 自动化：`rule_2_2_third_party_unregulated_no_operate`

## TC-03-05 规则3.1-依权益判定（非车险理赔样例）【P0】

- 数据：管制业务；G=是（国家金融监督管理总局）、H=是（用户个人信息无授权协议）、I/J=否
- 预期：规则 3.1；经营权="依权益判定"（恢复经营权后无 I/J 约束）
- 自动化：`rule_3_1_third_party_regulated_judges_after_restore`

## TC-03-06 规则3.1-存在约束 【P1】

- 数据：同上但 I=是（第三方商业机密，如"涉及供应商采购信息,未经三方许可禁止公开"）
- 预期：规则 3.1；网公司经营权=**无**；原因含"存在其他无经营权约束"

## TC-03-07 共享判定原因标准文案 【P0】

- 预期：所有规则的 reason 以"根据公司确权授权工作指引,在不违反法律法规合同约定前提下,网公司合法取得分公司及全资子公司所有数据相应权益"开头

## TC-03-08 与确权状态联动 【P1】

- 步骤：未保存表级清单时查看归集判定
- 预期：按 申请涉第三方标记 兜底判定；提示先完善表级清单
